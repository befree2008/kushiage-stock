# 炸串店库存管理 - 日常工作流

> 📋 这是老王和星月每周配合的标准操作流程

---

## 🔄 完整流程循环

```
┌───────────────────────────────────────────────────────────┐
│  1️⃣  每周固定日（比如周一）                                 │
│      星月提醒："老王，本周销售数据该更新了！"                │
└───────────────────────────────────────────────────────────┘
                             ↓
┌───────────────────────────────────────────────────────────┐
│  2️⃣  老王发美团管家导出的 Excel                              │
└───────────────────────────────────────────────────────────┘
                             ↓
┌───────────────────────────────────────────────────────────┐
│  3️⃣  星月自动处理：                                          │
│      ✅ 扣减库存（带日期防呆）                               │
│      ✅ 生成补货建议清单                                     │
└───────────────────────────────────────────────────────────┘
                             ↓
┌───────────────────────────────────────────────────────────┐
│  4️⃣  老王：看建议 → 下单 → 收货 → 拍进货单发我              │
└───────────────────────────────────────────────────────────┘
                             ↓
┌───────────────────────────────────────────────────────────┐
│  5️⃣  星月自动处理：                                          │
│      🔍 识别进货单 → 发表格给老王确认                        │
│      ✅ 确认后 → 自动入库更新库存                            │
└───────────────────────────────────────────────────────────┘
                             ↓
                         回到步骤 1
```

---

## 📅 详细步骤说明

---

### **第一步：导出销售数据（老王）**

**触发时机：** 每周固定日（比如周一早上），星月会主动提醒。

**老王操作：**
1. 登录美团管家后台
2. 导出上周（或指定日期范围）的销售统计 Excel
3. 发给星月

---

### **第二步：扣库存 + 补货建议（星月全自动）**

**系统自动完成：**

| 步骤 | 动作 | 防呆保护 |
|------|------|---------|
| 1 | 从 Excel 自动提取日期范围 | 日期不连续会提醒确认 |
| 2 | 美团菜品名 → SKU 智能匹配 | 未匹配会标注出来 |
| 3 | 按 SKU 汇总销量 | |
| 4 | 自动扣减对应库存 | 重复日期会强制阻止 |
| 5 | 按安全库存计算补货建议 | |

**老王会收到：**
```
📊 本周销售扣减完成！
├── 统计周期：2026-05-11 ~ 2026-05-17 (7天)
├── 扣减 SKU：61 个
└── 🛒 补货建议：
    SKU009 鸡肉小串     缺 3000 串 → 建议进 3 箱
    SKU005 玉米粒小串   缺 1500 串 → 建议进 2 箱
    SKU007 纸豆皮小串    缺 1000 串 → 建议进 1 箱
    ...
```

---

### **第三步：收货拍照（老王）**

**老王操作：**
1. 看星月给的补货建议，调整数量后下单
2. 货送到后，把进货单拍照发给星月

---

### **第四步：进货单识别 + 入库（星月全自动）**

**系统自动完成：**

| 步骤 | 动作 | 说明 |
|------|------|------|
| 1 | 🔍 视觉 AI 识别进货单图片 | 支持针式打印、折叠、折痕遮挡 |
| 2 | 📋 自动匹配 SKU，整理成表格发给老王确认 | 不确定的会标注"需确认" |
| 3 | ✅ 老王说"入库吧"，自动按箱规换算更新库存 | 每箱→串/kg/g/个自动换算 |
| 4 | 📝 完整记录操作日志 | 永久可追溯 |

**老王会收到确认表格：**
```
📦 进货单识别完成，共 11 件货品：
├── SKU039 卤油         1箱 = 10000g
├── SKU043 小串纸桶     1箱 = 500个
├── SKU036 紫苏孜然撒粉 1箱 = 10000g
├── SKU018 巴掌大鸡排   1箱 = 120串
├── SKU033 台湾无骨鸡柳 1箱 = 10kg
├── SKU014 开花肠       1箱 = 200串
├── SKU021 椒盐烧饼     1箱 = 160串
├── SKU005 玉米粒小串   1箱 = 1000串
├── SKU024 生炸鸡腿串   1箱 = 80串
├── SKU013 脆皮年糕     1箱 = 200串
└── SKU031 鸡叉骨       1箱 = 15kg

确认后我就更新库存！
```

## 🛡️ 安全防呆机制

| 场景 | 系统动作 |
|------|---------|
| **重复扣减**（同一日期范围扣了又扣） | ❌ 强制阻止，给出提示 |
| **日期不连续**（上次扣到5.10，这次从5.12开始） | ⚠️ 警告提醒，确认后继续 |
| **进货单有新品**（系统里没有的SKU） | ⚠️ 标注"需确认"，不自动入库 |
| **所有操作** | 📝 完整记录日志，随时可查 |

---

## 💡 老王小贴士

1. **进货单照片越清晰，识别越准**，拍的时候尽量平放、光线好
2. **日期范围尽量连续**，避免漏掉某几天的销量没扣
3. **有新品进货**提前告诉我，我把SKU加到字典里下次就能自动识别了
4. **想单独查库存/调账**随时说，我给你操作
